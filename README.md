# Guvi Internship Project
A professional Login and Profile management system.

## Tech Stack
* **Frontend:** HTML, Bootstrap, jQuery (AJAX)
* **Backend:** PHP
* **Database:** MySQL (Auth) & MongoDB (Profile Data)
* **Caching/Session:** Redis

## How to Setup
1. XAMPP mein Apache aur MySQL start karein.
2. MongoDB Compass connect karein.
3. Terminal mein `composer install` chalayein.
